# 🚀 TECSAA Digital Portfolio

A stunning 3D animated portfolio website for **TECSAA Digital** - One-Stop Solution for Digital Marketing.

## ✨ Features

- **3D Character Animation** - Interactive 3D model with smooth animations
- **Physics-based Tech Stack** - Floating service icons with realistic physics
- **Responsive Design** - Mobile-friendly, works on all devices
- **Modern UI/UX** - Smooth scrolling, custom cursor, professional layouts
- **SEO Optimized** - Meta tags and structured data ready

## 🛠️ Tech Stack

- **React 18** - UI Framework
- **Three.js** - 3D Graphics Engine
- **React Three Fiber** - React wrapper for Three.js
- **GSAP** - Advanced animations
- **Vite** - Build tool
- **TypeScript** - Type safety

## 📋 Contents

- Landing section with TECSAA branding
- About/Services section
- Interactive 3D tech stack visualization
- Project showcase (event.tecsaa.com, arriagaremodelingandpainting, catherinesoutsideeatery)
- Contact & Social links
- Smooth scroll animations

## 🚀 Quick Start

### Local Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

Visit `http://localhost:5173` to see the portfolio.

## 📤 Deployment to Vercel

### Option 1: Using Vercel CLI

```bash
# Install Vercel CLI
npm i -g vercel

# Deploy to Vercel
vercel
```

### Option 2: Using GitHub + Vercel (Recommended)

1. **Push to GitHub:**
   ```bash
   git remote add origin https://github.com/YOUR_USERNAME/tecsaa-portfolio.git
   git branch -M main
   git push -u origin main
   ```

2. **Deploy on Vercel:**
   - Go to [vercel.com](https://vercel.com)
   - Click "New Project"
   - Import the GitHub repository
   - Click "Deploy"
   - Done! Your site is live

## 📝 Customization

### Update Company Info
Edit these files:
- `src/components/Landing.tsx` - Main title & intro
- `src/components/Contact.tsx` - Contact details & social links
- `index.html` - Page title & meta tags
- `src/components/TechStack.tsx` - Services/tech labels

### Update Images
Replace tech stack images in `/public/images/` with TECSAA service logos.

### Update Colors
Edit CSS files in `/src/components/styles/` to match TECSAA brand colors.
